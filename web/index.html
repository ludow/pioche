Aide

Tracez un rectangle pour sélectionner une zone à rogner ; déplacez-le ou ajustez-le par ses poignées. Un clic ailleurs dans l'image efface la sélection.

Molette : zoom · double-clic : réajuster · Ctrl+glisser, clic molette ou Espace : déplacer l'image.

Maj+glisser : ajouter une zone (acte sur plusieurs colonnes) ; les zones sont assemblées en une seule image, dans l'ordre de traçage. Croix ou touche Suppr : supprimer une zone.

Acte sur deux pages : naviguez entre les pages, les zones tracées sur chaque page s'assemblent ensemble.
